常见的内存错误.
   1> 内存申请失败,但使用了.(new/malloc 立即判断ptr weather null)
   2> 没有initializtion memory  就使用了. (应立即初始化,避免将未初始化的内容作为右值使用.)
   3> 内存越界. (内存是连续的有下标的)
   4> 内存泄漏.(new/delete;malloc/free must match.)
   5> 内存释放之后,还继续使用了. (release memory ,should equal null)

static_cast 和 dynamic_cast区别.
   static_cast up conversion 安全的.下行不安全.(没有类型校验)
    dynamic_cast   up/down conversion 均安全的. 
   在进行上行转换是(子类->基类) static_cast 和 dynamic_cast 是一样的.
    下行转换时 dynamic_case 有类型校验,所以更安全.
